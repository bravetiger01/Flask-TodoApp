About MyToDoAPP

This website is a simple Todo List application built using Flask, a micro web framework for Python.

It was created as a learning project to explore the capabilities of Flask and to demonstrate a basic CRUD (Create, Read, Update, Delete) application.

The application allows users to create, read, update, and delete todo items, and also provides a search functionality to find specific todo items.

In addition to Flask, this application also uses Bootstrap for styling and layout, and Jinja2 for templating.

If you're interested in learning more about Flask or exploring other options for building web applications, I recommend checking out the following resources:

  • Flask Documentation
  • Bootstrap Documentation
  • Jinja2 Documentation
